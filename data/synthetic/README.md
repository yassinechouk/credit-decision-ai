# data/synthetic

Cas synthétiques pour démarrer sans données sensibles.

- Ce qu’il fait: fournit des exemples minimaux (ex: `sample_case.json`).
- Exemple simple:
```
{
  "case_id": "CASE_001",
  "monthly_income": 2000,
  "requested_amount": 5000,
  "documents": ["invoice_001.pdf"]
}
```
- Résultat attendu: l’API renvoie une décision stub et une explication.
